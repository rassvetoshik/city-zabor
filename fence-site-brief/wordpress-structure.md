# WordPress Structure

## Chto klient budet redaktirovat

- Teksty glavnoi
- Ceny "ot"
- Foto i galerei
- Kartochki napravleniy
- Portfolio
- FAQ
- Otzyvy
- Kontakty

## Post types

- `services`
  Dlya tipov zaborov, vorot, navesov i dopolnitelnykh uslug.

- `projects`
  Dlya portfolio rabot.

- `reviews`
  Dlya otzyvov.

- `faq`
  Dlya voprosov i otvetov.

## Taksonomii

- `service_category`
  Zabory, vorota, navesy, avtomatika, metallokonstruktsii.

- `project_category`
  Zabory, vorota, navesy.

## Polya dlya `services`

- Nazvanie
- Kratkiy podzagolovok
- Cena ot
- Edinitsa izmereniya
- Glavnoe foto
- Galereya
- Opisanie
- Spisok preimushchestv
- Kharakteristiki
- CTA-tekst knopki

## Polya dlya `projects`

- Nazvanie obekta
- Kategoriya
- Foto do/posle ili galereya
- Korotkoe opisanie
- Lokatsiya
- Vid raboty

## Polya dlya obshchikh nastroek

- Telefon
- WhatsApp
- Telegram
- Adres
- Region raboty
- Zagolovok pervogo ekrana
- Podzagolovok pervogo ekrana
- Tekst o garantii

## Chto ne dayem klientu lomat

- Dizayn i verstku
- Strukturu sektsiy
- Tekhnicheskie nastroyki WordPress
- Pluginy i kod temy
